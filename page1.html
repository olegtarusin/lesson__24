примеры и задания

  • какой результат первее попадет в консоль
  • function showName() {
    console.log('vasya');
    }
  • или
  • setTimeout(showName, 0); {
    console.log('kolya');
    }
  • Сначала был вывод vasya, a потом kolya
  • так происходит потому, что функции с обьявленной задержкой обрабатывает медленнее чем без нее, в нашем случае цыфра 0 -это обьявление задержки
  • верно ли задана функция
  • showMessage()
    let showMessage = function() {
    console.log('this is message');
    }
  • код не правельный , правельно будет:
  • let showMessage = function() {
    console.log('this is message');
    }
    showMessage()
  • в данном примере обьявление функции происходит в переменной , а до обьявления переменной ее не существует

  • как решить проблему
  • if (2>1) {
    function showMessage() {
    console.log('hello');
    }
    }
    showMessage()
  • ошибка возникает потому, что функция обьявлена только при соблюдении условия тоесть на втором уровне вложености, а вызывается на первом. проблему можно решить двумя способами
  • первое решение
  • if (2>1) {
    function showMessage() {
    console.log('hello');
    }
    showMessage()
    }
  • второе решение
  • function showMessage() {
    if (2>1) {

    }
    }
