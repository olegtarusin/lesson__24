обьектная модель документа

примеры и задания

  • получить в переменную элемент с атребутом data-say-hi и вывести значение этого атребута
  • <div data-say-hi="yes">привет</div>
  • решение
  • const data = document.querySelector('[data-say-hi]');
    let dataContent = data.innerHTML;
    console.log(dataContent);
  • вывод будет 'привет'
  • получить в переменную элемент с текстом йончи
  • <ul>
    <li>Корчи </li>
    <li>Иончи </li>
    </ul>
  • решение
  • let data2 = document.querySelector ('ul.homework__exemple');
    data2 = data2.lastElementChild;
    data2 = data2.innerHTML;
    console.log(data2);
  • вывод Иончи
  • получить в перемеенную коллекцию с классом like
  • <div class="like"></div>
    <div class="subscribe"></div>
    <div class="like subscribe"></div>
  • решение
  • let data3 = document. querySelectorAll('.like');
    for (let data3Elem of data3) {
    console.log(data3Elem);
    }
  • получаем элементы колекции <div class="homework__exemple like">text</div> <div class="homework__exemple like subscribe">text</div>
  • куда добавится елемент <li>текст </li>
  • const list = document.querySelectorAll('ul');
    list.insertAdjacentHTML(
    `beforeend`,
    `<li>текст</li>`
    );
  • HTML
  • <ul>
    <li>пункт1 </li>
    <li>пункт2 </li>
    </ul>
  • разберем пример на тегах второго задания
  • let data4 = document.querySelector ('ul.homework__exemple');
    data4.insertAdjacentHTML(
    "beforeend",
    '<li>текст beforeend</li>'
    );
  • текст попадет внутрь тега после последнего элемента
